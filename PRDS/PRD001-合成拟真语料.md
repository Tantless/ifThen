# PRD001-合成拟真语料

## prd-合成拟真语料-完成情况

### 目标

为真实性路线图 `realism-01` 先补齐可提交、可导入、可评估的合成拟真长消息样本，避免直接使用真实隐私聊天记录。

### 需求

- 生成 3 段每段 1000+ 条的 QQChatExporter 兼容私聊记录。
- 三段分别覆盖隐藏心理阴影导致告白失败、冲突修复、错过窗口三类关系场景。
- 每段都要包含至少 3 个关键可改写点、至少 1 个 cutoff 后才揭示的客观事实，并明确 modeler-only evidence 边界。
- 每段都要补齐 `conversation.txt`、`timeline.md`、`rewrite-points.md`、`truth-after-cutoff.md` 和 `generation-notes.md`。
- 所有合成记录必须避免真实姓名、真实学校、真实联系方式等可识别信息。

### 验收标准

- 三段 `conversation.txt` 均能被 `parse_qq_export()` 解析。
- 每段解析后消息数不少于 1000，header 的 `消息总数` 与解析数量一致。
- 每段至少包含 3 个关键可改写点，且标注 cutoff 前判断与 modeler-only evidence 判断的差异。
- 每段至少能完成导入、分析，并发起当前 `single_reply` 或 `short_thread` 推演。
- 不提交 API key；生成所需密钥只来自本地环境变量或运行时配置。
